# Ejercicios resueltos

Colección inicial de ejercicios con pistas y soluciones comentadas para reforzar
los artículos y cuadernos del tutorial.

## Cómo usar esta sección

Conviene intentar primero los ejercicios de cada artículo o cuaderno. Después se
puede consultar la pista, y solo al final la solución completa.

El objetivo no es memorizar respuestas, sino aprender a reconstruir el
razonamiento.
