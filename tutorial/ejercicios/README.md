# Ejercicios

Espacio para recopilar ejercicios del tutorial.

## Organización prevista

- Ejercicios por módulo.
- Problemas conceptuales.
- Problemas de cálculo.
- Demostraciones guiadas.
- Retos de ampliación.

## Formato recomendado

Cada ejercicio debería indicar:

- módulo relacionado;
- dificultad estimada;
- enunciado;
- pistas opcionales;
- solución o esquema de solución cuando proceda.
