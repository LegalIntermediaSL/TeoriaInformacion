# 04 - Complejidad computacional

Módulo sobre la cantidad de recursos necesarios para resolver problemas
computacionales.

## Objetivos

- Analizar el coste temporal y espacial de algoritmos.
- Introducir clases de complejidad.
- Comprender reducciones polinómicas y completitud.
- Estudiar la diferencia entre tratabilidad e intratabilidad.

## Contenidos previstos

- Tiempo y espacio como recursos.
- Modelos de coste.
- Clases P y NP.
- Verificación eficiente.
- Reducciones polinómicas.
- NP-completitud.
- Problemas clásicos NP-completos.
- Complejidad espacial.
- Jerarquías de complejidad.

## Ejemplos previstos

- Análisis de algoritmos simples.
- Problemas SAT, CLIQUE y SUBSET-SUM.
- Reducciones entre problemas.
- Comparación entre búsqueda exhaustiva y verificación.
